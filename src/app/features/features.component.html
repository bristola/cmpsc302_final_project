Features

Azariel Technologies LLC have developed a suite of software for the cryptocurrency markets. By using technology such as Python and Amazon Web Services, we are able to aid the trading of cryptocurrency.

Market Analysis

One software system which is in production is a market analysis tool. We are constantly collecting market price data on a variety of crypto coins. Using constantly evolving data, we can analyze trends, regressions, drops, and much more. All data is saved to a database which can be connected to.

Order Book Analysis

Much like our market analysis tool, we have similar software which analyzes orders. Our system is constantly looking at the new orders coming into the order book, so you can get instant feed back about the different orders that are relavent to trading. This system is constantly updating live, and can be queried in our database as well.

Account Balancing

With software which is constantly making a large amount of trades a day, we need some system in place to evaluate its performance. After a many trades, the rounding errors which must be done can set account money balances slightly off from desired. This software makes sure that your account is always balanced to your liking.

Automated Trading

Our main offering at Azariel Technologies is automated trading. Using a proprietary algorithm developed in Python, we can constantly make trades on the cryptocurrency market for profit. By simply setting different variables for how much money you want to trade with, our system will analyze the markets and make trades for profit automatically. All of our software systems and features work together to produce a high quality cryptocurrency trading system.
